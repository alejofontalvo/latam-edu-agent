# Guía para Presentar Visualmente el Proyecto

## Guion recomendado

1. Abrir el dashboard principal y mostrar KPIs globales.
2. Explicar que los datasets están prealimentados por administrador.
3. Abrir Observatorio LATAM y mostrar países/pruebas disponibles.
4. Analizar un dataset: KPIs, filtros y gráficos.
5. Ir a Comparativos y mostrar brechas por país o género.
6. Abrir Chat IA y hacer una pregunta analítica.
7. Mostrar documentos RAG como biblioteca metodológica.
8. Generar una evaluación tipo prueba estatal.
9. Mostrar endpoints y exportaciones Power BI.
10. Cerrar explicando Docker/n8n como mejoras futuras.

## Mensaje clave

LATAM EduAgent no es un simple cargador de archivos: es un observatorio educativo con IA, datos normalizados, análisis comparativo y preparación para BI.
